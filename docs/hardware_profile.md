## 6. 安装稳定性检查清单

完成机械装配后，逐项打勾：

- [ ] 摄像头支架在 5N 横向推力下偏移 ≤ 0.5 mm
- [ ] 摄像头排线无折角、无张力
- [ ] LCD 排线已固定，不会被震动拉脱
- [ ] 车体翻倒角 ≤ 60° 时，摄像头不直接撞地（plan §0 任务文档约束）
- [ ] 摄像头光轴与车体 x 轴对齐误差 ≤ 1°（用激光指示器投影法验证）
- [ ] 静态俯仰角与设计值偏差 ≤ 1°
- [ ] 摄像头视野下沿不被车体/电池/线缆遮挡

## 7. 与后续阶段的标定衔接

| 标定 | 何时执行 | 输出 | 文件 |
|------|----------|------|------|
| 内参（一次性） | 装配完成立即做 | `K, dist` | PC 端生成 → `calib.json` |
| IPM 外参（阶段 C） | 每次支架变动后 | `H, ipm_lut` | `tools/calibrate_ipm.py` |
| 光度（阶段 B） | 每次场地光照变化 | `line_threshold` | `tools/calibrate_photometric.py` |
| 运动学（主控侧） | 出车前一次性 | 轮径 / PPR / 死区 | 主控 Flash |
